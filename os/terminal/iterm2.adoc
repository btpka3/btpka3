## 竞品

深度思考
macOS 上主流的第三方终端选择还挺多的，按风格大致分几类：

* GPU 加速 / 高性能型
** Kitty — GPU 渲染，速度极快，配置用纯文本文件，支持图片显示、ligatures，扩展通过 "kittens" 实现。跨平台，开源免费。配置文件非常灵活，适合喜欢折腾的人。
** Alacritty — 同样 GPU 加速，号称最快的终端模拟器。极简哲学，没有 tab/分屏（推荐配合 tmux 使用），用 TOML/YAML 配置。开源免费，跨平台。
** WezTerm — GPU 加速，用 Lua 脚本配置，功能很全：内置 multiplexer（不需要 tmux）、tab/分屏、ligatures、图片支持。作者很活跃，开源免费。
** Ghostty — 比较新，Zig 编写，Mitchell Hashimoto（HashiCorp 创始人）的项目，主打原生体验和极致性能，macOS 上用 Metal 渲染。目前还在积极开发中。

* 经典 / 功能丰富型
** iTerm2
** Hyper — Vercel 出品，基于 Electron（Web 技术栈），可以用 JS/CSS 做主题和插件，颜值很高，但性能不如 GPU 加速的那几个，内存占用也偏大。
** Warp — 比较新的终端，内置 AI 辅助、命令块（block）概念，每条命令的输出是独立的块，可以分享和协作。界面现代，有免费版和付费版。不过它需要登录账号，隐私方面有争议。
** Rio — Rust 编写，GPU 加速，支持 WebGL 着色器做视觉效果，比较新但在快速发展。
