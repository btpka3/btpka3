* link:https://github.com/BurntSushi/ripgrep[ripgrep]
* link:https://pkgs.alpinelinux.org/package/edge/community/x86_64/ripgrep[alpine : ripgrep]

使用 Rust 编写的跨平台文本搜索工具, 可 递归搜索目录。


[source,shell]
----
# macos 上安装
brew install ripgrep
# ubuntu 上安装
sudo apt-get install ripgrep
# alpine 上安装
apk add ripgrep

ug --version
ug --help

----
