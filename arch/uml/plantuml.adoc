== VS code


* 安装 vs code 插件 PlantUML（作者：jebbs）
* 可选：安装 vs code 插件 Markdown Preview Mermaid Support（可选，配合使用）。
该插件是渲染 link:https://mermaid.ai/web/[Mermaid] 的，而非 plantuml.
* 如果选择默认 内置 markdown 预览插件，则需要安装  link:https://mermaid.ai/web/[Mermaid] 的 VsCode 插件才行，但那又是另外一套 uml 语法了




* 安装 Graphviz（同之前说明）： `brew install graphviz`

* VSCode
** 打开设置: kbd:[Cmd/Ctrl + ,]
** 搜索 plantuml
** 配置 Plantuml: Render 为 PlantUMLServer 或 Local
** 如选 Local，需设置 Plantuml: Jar 路径（插件会自动下载）
** 如果选择 PlantUMLServer， 则需要配置  plantuml.server
+
----
{
  "plantuml.render": "PlantUMLServer",
  "plantuml.server": "http://www.plantuml.com/plantuml"
}
----

=== 适用场景
* 选择 PlantUML 的场景
** ✅ 需要完整的 UML 2.5 规范支持
** ✅ 企业级架构设计文档
** ✅ 复杂的类图、组件图、部署图
** ✅ 需要精细控制布局和样式
** ✅ Java/Kotlin 生态系统项目
** ✅ 需要生成多种格式（包括 ASCII）

* 选择 Mermaid 的场景
** ✅ Web 项目文档
** ✅ GitHub/GitLab README
** ✅ 快速原型设计
** ✅ 团队非技术人员参与
** ✅ 需要甘特图、饼图、思维导图
** ✅ 零配置、开箱即用
** ✅ 现代前端技术栈
