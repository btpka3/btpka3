Terraform 是一款 基础设施即代码（IaC） 工具，用于安全高效地构建、变更和版本化云及本地基础设施。通过声明式配置文件定义目标状态，自动对比并执行最小化变更。

核心 CLI ： 	是 Go（主程序 terraform）开发
配置语法：	HCL（HashiCorp Configuration Language，类 JSON/YAML 的声明式 DSL）
Provider SDK：	Go（官方推荐）、Python/Java 等（社区插件）
状态文件：	JSON 格式（可加密存储）


== 收费模式

v1.6.x 及之前为 MPL 2.0 开源协议；v1.7+ 起改为 BSL 1.1（非纯开源）。


BSL 限制：禁止直接提供托管 Terraform 服务（如竞品 SaaS），但企业内部自用完全免费

商业版： HCP Terraform（原 Terraform Cloud）提供协作、策略检查、远程状态管理等增值服务，按用户/资源量订阅
企业本地部署	: Terraform Enterprise 支持私有化部署，需商业授权


== 局域网/离线环境使用

核心 CLI 可在无外网环境运行

* 使用 terraform init -backend-config 指定本地状态后端
* 搭建私有 Registry（如 hashicups 或自研镜像站）缓存 Provider/Module
* 禁用遥测：export CHECKPOINT_DISABLE=1
* 离线安装 Provider：手动下载至 ~/.terraform.d/plugins/
