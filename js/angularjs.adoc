==== 使用 WebStorm/ IDEA 自动补全功能

[arabic]
. 下载 https://github.com/johnpapa/angular-styleguide/blob/master/a1/assets/webstorm-angular-live-templates/webstorm-angular-live-templates.xml?raw=true[webstorm-angular-live-templates.xml]
. 将下载的xml文件移动到 `~/.IntelliJIdea15/config/templates/` 目录下
. 重启 IDEA Intellj
. 之后就可以在编辑 js 文件时， 通过 `ngapp++<++tab++>++` 来快速使用模板了。

==== $http 提交 form data

[source,js]
----
controller: ['$http', "$httpParamSerializerJQLike", function ($http, $stateParams, $httpParamSerializerJQLike) {
    // 微信:统一下单, 并准备JS API 支付所需的参数
    $http.post(appConfig.apiPath + '/testZll/testBuy', {
        count: $scope.count,
        price: $scope.price
    }, {
        headers: {
            'Content-Type': 'application/x-www-form-urlencoded; charset=UTF-8'
        },
        transformRequest: $httpParamSerializerJQLike
    }).then(function (resp) {
        //
    });
}]
----
