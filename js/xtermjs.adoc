== 链路
[source,plain]
----
方案： 浏览器(xterm.js)  -> spring boot (仅WebSocket) -> nodeJs: node-pty  -> 操作系统 Shell
----

关键注意事项
* 多用户隔离：每个 WebSocket 连接对应一个独立的 PTY 进程，不能共享
* 窗口大小同步：前端 resize 时要通过 WebSocket 通知后端调用 ioctl(TIOCSWINSZ) 调整 PTY 尺寸
* 安全：暴露 shell 等于给服务器开了后门，必须加认证（Spring Security + WebSocket 握手拦截），建议限制可执行的命令范围
* 连接保活：设置 WebSocket 心跳，超时自动关闭 PTY
* 字符编码：确保前后端都使用 UTF-8
