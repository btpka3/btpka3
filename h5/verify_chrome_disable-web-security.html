Chrome --disable-web-security 验证工具

启动方式

请使用以下命令启动 Chrome（需先关闭所有 Chrome 窗口）：

open -na "Google Chrome" --args --disable-web-security --user-data-dir="/tmp/chrome-nosec"

然后在新开的 Chrome 中打开本文件，点击下方按钮测试。
