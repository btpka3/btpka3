## icon font

link:https://lucide.dev/icons/[lucide]

Lucide 是一个开源的图标库，它是 Feather Icons 的社区 fork 和延续。以下是核心介绍：

基本概念 Lucide 提供了一套风格统一、线条简洁的 SVG 图标集合，图标数量超过 1000 个，覆盖了 UI 开发中绝大多数常见场景（导航、文件、用户、通信、媒体、开发工具等）。

主要特点

* 一致的视觉风格：所有图标基于 24x24 的 viewBox，使用 2px 描边宽度，圆角和端点统一，确保整体视觉协调
* 多框架支持：官方提供了 lucide（原生）、lucide-react、lucide-vue、lucide-vue-next、lucide-svelte、lucide-preact、lucide-solid 等包，几乎覆盖主流前端框架
* Tree-shakable：支持按需导入，只打包你使用的图标，不会引入完整图标集
* 可定制：可以通过 props 控制 size、color、strokeWidth 等属性
* MIT 许可证：免费商用
