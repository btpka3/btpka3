== 数据泄露查询类
Firefox Monitor（monitor.mozilla.org）

Mozilla 提供的免费服务，底层数据实际上就来自 Have I Been Pwned 的 API。它会告诉你邮箱涉及哪些泄露事件，并提供安全建议。

DeHashed（dehashed.com）

提供数据泄露查询服务，但需要付费。它可以搜索邮箱、用户名、IP 地址等在泄露数据库中的记录，覆盖面比较广。

LeakCheck（leakcheck.io）

类似于 HIBP 的泄露查询服务，提供免费的有限查询和付费 API。支持邮箱、用户名等维度的查询。

== 密码安全类
1Password Watchtower / Password Checker

一些密码管理器（如 1Password、Bitwarden、LastPass）内置了密码泄露检查功能，会将你保存的密码与已知泄露的密码库进行比对。

Google Password Checkup

Chrome 浏览器和 Google 账号内置的密码检查功能，会自动检测你保存的密码是否出现在已知泄露数据中。

==  威胁情报与安全监控类
VirusTotal（virustotal.com）

Google 旗下的免费在线服务，用于检测文件（病毒、木马）和 URL 是否属于恶意内容。安全从业人员广泛使用。

AbuseIPDB（abuseipdb.com）

提供 IP 地址的信誉查询服务。可以查询某个 IP 是否被标记过恶意行为（如暴力破解、DDoS、垃圾邮件等），适合用来排查日志中的可疑 IP。

Shodan（shodan.io）

被称为"搜索引擎版的物联网"，可以搜索互联网上所有对外开放的设备和端口。安全研究者用它来发现暴露在外网的摄像头、路由器、服务器等。

Censys（censys.io）

与 Shodan 类似，用于搜索互联网上的主机和设备，常用于安全审计和资产发现。

ThreatFox（threatfox.abuse.ch）

由 abuse.ch 运营的免费威胁情报平台，分享恶意 IP、URL、域名等 IoC（Indicators of Compromise）。

== 暗网与泄露数据监控类
DarkOwl（darkowl.com）

提供暗网数据监控服务，帮助企业监控自己的域名、邮箱是否在暗网论坛或市场中被提及。

Recorded Future

企业级威胁情报平台，整合了公开来源、暗网和私有数据，提供全面的安全威胁情报。

== 其他实用安全工具
URLScan（urlscan.io）

免费的 URL 扫描和分析服务，可以预览网页内容、检测恶意链接，常用于判断一个 URL 是否安全。

Hybrid Analysis（hybrid-analysis.com）

免费的恶意文件在线分析服务，上传可疑文件后会自动进行沙箱分析并生成详细报告。

SecurityTrails（securitytrails.com）

提供 DNS 历史记录、WHOIS 查询、子域名发现等安全侦察功能，常用于渗透测试和安全审计。

这些服务各有各的侧重点，如果只是个人用户关注自己的账号安全，Firefox Monitor 和浏览器自带的密码检查就足够了。如果是企业安全运营，通常会结合 AbuseIPDB、VirusTotal、Shodan 等做更全面的威胁情报收集和资产监控。
