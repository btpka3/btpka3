

### pyenv
解释器版本管理：安装/切换不同 Python 主版本（如 3.8 ↔ 3.12）→ pyenv, asdf, uv（实验）

pyenv 是 解释器版本管理（事实标准）。  但 本身不建虚拟环境（需插件）。编译安装依赖系统库，

[source,shell]
----
brew install pyenv
brew install pyenv-virtualenv

cat <<EOF > ~/.zshrc
export PYENV_ROOT="$HOME/.pyenv"
[[ -d $PYENV_ROOT/bin ]] && export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init - zsh)"
EOF

# 安装 pyenv-virtualenv 插件
#git clone git@github.com:pyenv/pyenv-virtualenv.git $(pyenv root)/plugins/pyenv-virtualenv
cat <<EOF > ~/.zshrc
eval "$(pyenv virtualenv-init -)"
EOF

# 查看帮助文档
pyenv help
pyenv help global
# 列出本地已经安装的 python 版本
pyenv versions
# 检查当前环境使用的 python 版本
pyenv version

# 列出可安装的版本列表
pyenv install --list
# 安装一个给定版本的 python
pyenv install 3.13
# 设置全局版本
pyenv global 3.13

# 查看已经安装的版本
pyenv versions

pyenv help global
# 安装 指定的 python 版本
pyenv install 3.11
# 用指定的 python 版本 创建一个 虚拟环境
pip install virtualenv
pyenv virtualenv 3.11 myenv


# 当前shell使用指定的环境
pyenv activate myenv
# 全局使用给定的环境
pyenv global myenv
