



## virtualenv

虚拟环境管理：为项目创建隔离依赖环境 → venv, virtualenv, conda, poetry, uv

virtualenv, venv :

* link:https://virtualenv.pypa.io/en/latest/[virtualenv]


error: externally-managed-environment



[source,shell]
----
brew install virtualenv
#pipx install virtualenv
#pip3 install virtualenv

# 到自己的项目目录，将 `.venv` 添加到 .gitignore 文件里，
cd first-spring-boot3
# 创建 .venv 目录, 该目录下会有 pyvenv.cfg 文件
python3 -m venv .venv
# 使用该环境
source .venv/bin/activate
# 不使用该环境
deactivate
# 该环境下更新 setuptools
pip3 install --upgrade setuptools
pip3 install --upgrade pip
# 安装相关 python 依赖
pip3 install torch===2.5.1
 torchvision torchaudio
pip3 install vllm
