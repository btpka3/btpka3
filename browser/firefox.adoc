== 允许 https 和 http 内容混合

about:config -++>++ security.mixed++_++content.block++_++active++_++content : false

PS : chromium 浏览器中允许混合内容：在URL地址栏最后，点击盾牌按钮，点击 "`allow unsafe javascript`"
