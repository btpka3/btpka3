== 全局安装

[source,shell]
----
#brew install chroma

curl -sSL https://raw.githubusercontent.com/chroma-core/chroma/main/rust/cli/install/install.sh | bash

# 方式一：推荐
uv tool install chromadb

# 方式二： 使用虚拟环境
# 创建虚拟环境
python3 -m venv ~/.venvs/chroma
source ~/.venvs/chroma/bin/activate

# 然后在虚拟环境中安装
pip install chromadb


----

== 安装-作为库
[source,shell]
----
# 通过 rust 的 cargo 安装
cargo add chroma

mkdir /tmp/chroma_store
chroma run --path /tmp/chroma_store
----
