=== 简介

** https://etcd.io/docs/v3.3/dev-guide/limit/[System limits] 默认单个请求最大 1.5MB ，可调。

== 安装

=== macos

[source,shell]
----
brew install etcd
# 第一个窗口
etcd

# 第二个窗口
etcdctl put greeting "Hello, etcd"
etcdctl get greeting
----
