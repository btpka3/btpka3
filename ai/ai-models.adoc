== 补充说明

=== 上下文长度说明
- **1M tokens** 约等于 75 万英文单词或约 50-70 万中文字符
- 实际可用上下文受 API 定价和订阅层级限制
- 部分模型的"数百万 tokens"为理论值，实际使用受硬件限制

=== 开源定义
- **是（开源权重）**: 模型权重可下载，可本地部署，但可能有许可限制
- **部分开源**: 部分模型或尺寸开放，或仅限研究用途
- **否**: 闭源，仅通过 API 或 Web 界面访问

=== Thinking 模式
- **GPT o 系列**、**Claude**、**DeepSeek-R1**、**Qwen 3** 均支持显式推理模式（Thinking Mode），在数学、代码、逻辑推理场景中表现更优

=== 多模态能力
- 原生支持图片/视频/音频的模型：**GPT-4o**、**Gemini 2.5**、**Llama 4**、**Pixtral**、**GLM-4**
- 纯文本模型需配合额外模块才能处理非文本输入
