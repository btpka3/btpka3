



## ollama

Ollama 是一个开源项目，它旨在简化在本地运行和管理大语言模型（LLMs）的过程。通过Ollama，用户可以在自己的计算机上部署和使用各种大语言模型，而不需要依赖云服务。

link:https://ollama.com/[ollama]

=== /tools/list

[source,json]
----
{
  name: "searchFlights",
  description: "Search for available flights",
  inputSchema: {
    type: "object",
    properties: {
      origin: { type: "string", description: "Departure city" },
      destination: { type: "string", description: "Arrival city" },
      date: { type: "string", format: "date", description: "Travel date" }
    },
    required: ["origin", "destination", "date"]
  }
}

== skills
Skills是2025年10月由Anthropic在Claude Code上推出的特性，后于同年12月18日作为标准开放，迅速在AI开发者社区引发关注。该技术允许将流程性知识封装为可复用的能力包，供Agent调用执行任务，形式上是一个包含SKILL.md、脚本、模板等资源的文件夹，而非单一文本指令。


link:https://code.claude.com/docs/en/plugins-reference#skills[Claude code: plugins reference : skills]
