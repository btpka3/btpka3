n8n 工作流引擎 , n8n（读作 "n-eight-n"，即 "nodemation" 的缩写）是一个开源的工作流自动化平台。


核心概念
它是一个基于节点（node-based）的可视化工具，让用户通过拖拽方式将不同的应用、API 和服务连接起来，构建自动化工作流。每个节点代表一个操作步骤，节点之间的连线定义了执行顺序和数据流向。


* 主要特点
** 可视化编排：通过拖拽节点构建工作流，无需编写代码即可完成大部分自动化任务
** 丰富的集成：内置数百种预置节点，支持各种 SaaS 服务（如 Slack、GitHub、Google Sheets、数据库、HTTP 请求等）
** 自托管优先：可以部署在自己的服务器上，数据完全可控，同时也提供云服务版本
** 代码扩展能力：在节点中支持编写 JavaScript/TypeScript 代码，处理复杂逻辑
** 触发器机制：支持定时触发、Webhook 触发、事件触发等多种方式启动工作流
** 开源协议：采用"可持续使用许可证"（Sustainable Use License），个人和内部商业使用免费

* 典型应用场景
** 数据同步与迁移（如 CRM 到数据库）
** 通知与告警自动化
** API 编排与数据转换
** AI Agent 工作流编排（近年来新增的 AI 相关节点）
** 表单处理与审批流程
* 类似工具的比较
和 Zapier、Make（原 Integromat）类似，但 n8n 的核心差异在于可以自托管，并且源代码开放，更适合有技术团队、对数据隐私有要求的企业。近年来它也增加了对 AI/LLM 集成的支持，可以用来编排多 Agent 工作流。

简单来说，n8n 就是一个"把重复性操作串起来自动执行"的工具，用图形化界面代替手写脚本。
