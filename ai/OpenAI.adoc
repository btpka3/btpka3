=== 根据 response_id 获取模型响应

GET /responses/{response_id}

[source,shell]
----
curl ${OPENAI_BASE_URL}/responses/resp_abc123 \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer $OPENAI_API_KEY"
----


=== 根据 response_id 删除模型响应

DELETE /responses/{response_id}

[source,shell]
----
curl -X DELETE ${OPENAI_BASE_URL}/responses/resp_abc123 \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer $OPENAI_API_KEY"
----

  -H "Content-Type: application/json" \

=== 获取给定输入的 token 数量
POST /responses/input_tokens

[source,shell]
----
curl -X POST ${OPENAI_BASE_URL}/responses/input_tokens \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer $OPENAI_API_KEY" \
    -d '{
      "model": "gpt-5",
      "input": "Tell me a joke."
    }'
----

[source,json]
----
{
  "object": "response.input_tokens",
  "input_tokens": 11
}
----


=== 终止响应

POST /responses/{response_id}/cancel

[source,shell]
----
curl -X POST ${OPENAI_BASE_URL}/responses/resp_123/cancel \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer $OPENAI_API_KEY"
----

=== 将输入文本转换成大模型理解的 高维数值向量

/embeddings

输出: 数学向量（浮点数组）
用于: 检索、相似度、聚类、路由

=== 对话


[cols="1,2,2"]
|===
| 维度 | +/v1/chat/completions+ | +/v1/completions+
| *定位* | 多轮对话 / 指令遵循 / 智能体交互 | 单轮文本续写 / 基础语言建模
| *输入结构* | +messages+ 数组（含 +role+ 字段） | +prompt+ 字符串或字符串数组
| *支持模型* | GPT-3.5-Turbo、GPT-4、GPT-4o、o1/o3 等所有现代模型 | 仅旧版 Base 模型（如 +ada+、+babbage+、+curie+、+davinci+、+davinci-002+）
| *高级能力* | ✅ 系统提示词、工具调用、视觉输入、JSON模式、结构化输出、多模态 | ❌ 无原生支持，需手动拼接特殊 Token 模拟
| *官方状态* | 🟢 *强烈推荐使用*，持续迭代优化 | 🔴 *已弃用（Legacy）*，不再上架新模型，旧模型将逐步下线
| *典型场景* | 聊天机器人、RAG问答、Agent工作流、代码助手 | 历史项目兼容、简单文本续写、老模型微调推理
|===
