== 可编程的 AI 浏览器自动化框架 (终极解决方案)


* Browser Use (基于 LangChain/LlamaIndex)

link:https://github.com/browser-use/browser-use[browser-use]

link:https://browser-use.com/[browser-use.com], 收费。


原理：它是一个开源项目，允许大语言模型（LLM）直接控制浏览器（通过 Playwright 或 Puppeteer）。
能力：
你可以给 AI 一个自然语言指令：“登录这个网站，找到网络请求中返回用户 ID 的那个接口，并把数据存下来。”
AI 会自动打开浏览器，执行登录，监听 Network 事件，解析返回的 JSON，并提取数据。
它能完美处理动态 DOM，因为它是像真人一样操作浏览器，等待元素加载后再分析。
技术栈分析：由于它运行在无头浏览器环境中，可以拦截所有流量，轻松分析出网站的反爬策略（如 Cloudflare, Akamai）和技术指纹。


*  Playwright + Codegen + AI 插件

Playwright 自带的代码生成器可以记录你的操作。结合 VS Code 的 AI 插件（如 GitHub Copilot），你可以录制一段操作，然后让 AI 根据录制的轨迹（包含所有的 Network 请求和 DOM 变化）优化代码，自动处理登录态维持和数据解析。
