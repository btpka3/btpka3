== claude

只能针对单个工程项目。

[source,shell]
----
cd xxxProjectDir
# 仅仅为d当前工程 为 claude cli 增加 openspec 相关 command/skill
openspec init --tools claude

# 会为当前工程增加以下扩展：
.claude/commands/opsx/apply.md
.claude/commands/opsx/archive.md
.claude/commands/opsx/explore.md
.claude/commands/opsx/propose.md
.claude/skills/openspec-apply-change
.claude/skills/openspec-archive-change
.claude/skills/openspec-explore
.claude/skills/openspec-propose
----

在 claude cli 中可以通过相关 斜杠 命令进行相关操作。

[source,shell]
----
/opsx:propose	# 生成规范提案（包括 specs、设计文档、任务列表）
/opsx:apply	    # 执行实施任务，根据已批准的规范进行编码
/opsx:archive	# 归档已完成的变化，更新规范文档

# 保存在项目根目录的 的 openspec/specs/ 目录下
----
