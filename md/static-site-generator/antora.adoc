* link:https://antora.org/[antora] 对 asciidoc 构建静态网站。


[source,shell]
----
pnpm add -g @antora/cli @antora/site-generator @antora/lunr-extension --registry=https://registry.npmjs.org
antora --version
antora --help

antora antora-playbook.yml --to-dir ./dist --cache-dir ./antora-cache
----

== antora-playbook.yml
[source,yaml]
----
site:
  title: Docs Site
  start_page: component-b::index.adoc
content:
  sources:
  - url: https://gitlab.com/antora/demo/demo-component-a.git
    branches: HEAD
  - url: https://gitlab.com/antora/demo/demo-component-b.git
    branches: [v2.0, v1.0]
    start_path: docs
ui:
  bundle:
    url: https://gitlab.com/antora/antora-ui-default/-/jobs/artifacts/HEAD/raw/build/ui-bundle.zip?job=bundle-stable
    snapshot: true
----

== search

[source,plain]
----
┌────────────┬───────────────────────────┬───────────────┬───────────────────────────────────────┐
│    方案    │         索引大小          │   中文支持    │                 特点                  │
├────────────┼───────────────────────────┼───────────────┼───────────────────────────────────────┤

├────────────┼───────────────────────────┼───────────────┼───────────────────────────────────────┤
│ Pagefind   │ 很小（分片加载）          │ 原生支持      │ WASM 驱动，按需加载索引片段，适合大站 │
├────────────┼───────────────────────────┼───────────────┼───────────────────────────────────────┤
│ FlexSearch │ 中等                      │ 需配置        │ 极快的内存搜索，支持模糊匹配          │
├────────────┼───────────────────────────┼───────────────┼───────────────────────────────────────┤
│ Fuse.js    │ 较大                      │ 无需分词      │ 模糊搜索，无需预建索引，适合小数据量  │
├────────────┼───────────────────────────┼───────────────┼───────────────────────────────────────┤
│ MiniSearch │ 中等                      │ 需配置        │ 轻量零依赖，API 友好                  │
├────────────┼───────────────────────────┼───────────────┼───────────────────────────────────────┤
│ Orama      │ 中等                      │ 原生支持      │ 现代全文搜索引擎，支持 typo tolerance │
├────────────┼───────────────────────────┼───────────────┼───────────────────────────────────────┤
│ Tinysearch │ 极小（WASM+bloom filter） │ 有限          │ 索引体积最小，但功能简单              │
└────────────┴───────────────────────────┴───────────────┴───────────────────────────────────────┘
