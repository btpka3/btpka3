https://www.mintlify.com/


mintlify 是一个面向团队的智能知识平台（Intelligent Knowledge Platform），核心定位是帮助企业和开发者团队构建"面向人类和 AI 都能阅读"的文档体系。


它的主要特点包括：

. 现代化的文档体验

基于 MDX 格式，内置了精致的默认主题，支持搜索、导航、代码高亮、暗色模式等开箱即用的功能，视觉效果接近 Stripe、Linear 级别的产品文档。

. AI 原生能力

这是它区别于传统静态站点生成器（如 Docusaurus、VitePress）的关键。Mintlify 深度整合了 LLM 能力，比如自动生成文档摘要、AI 搜索问答、以及能够"自我更新"的文档 Agent。

. API 文档自动生成

它能直接基于 OpenAPI/Swagger 规范自动渲染交互式 API 参考文档，支持代码示例自动生成，对开发者产品非常友好。

. 企业级安全与协作

提供权限管理、SSO、私有部署等企业级功能，适合中大型团队的知识管理需求。

❌ Mintlify 不是开源的，也没有官方支持的本地离线版本，有提供 CLI（npx mintlify dev）做本地预览，但也要先上传到他们的 SAAS服务器上翻译。故不适用于 企业内部文档、私有部署场景。
