

### docker service

```bash
docker service create --publish     # 创建一个服务。服务的端口映射到节点主机的端口
docker service inspect              # 显示一个或多个服务的详情
docker service logs                 # 获取给定服务的日志
docker service ls                   # 列出所有的服务
docker service ps                   # 列出给定服务的任务列表
docker service rm                   # 删除一个或多个服务
docker service scale                # 扩容或缩减一个服务的启动副本数
docker service update               # 更新一个服务
```
