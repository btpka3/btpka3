# vagrant vs. docker
- vagrant 解决 (1) 开发环境快速部署 (2) 开发环境更迭。 底层就是一个安装了 Linux的 VirtualBox 虚拟机。
- vagrant 的出现早于 docker 容器
结论：直接忽略 vagrant，关注 docker 吧
