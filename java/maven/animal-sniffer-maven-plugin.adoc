使用高版本jdk（jar包）时，检查是否使用了 低版本的 jdk/jar 中没有的 api。
用于使用包版本编程，但用低版本运行。


[source,shell]
----
# 检查是否使用了 jdk 1.8 的没有的API
mvn animal-sniffer:check \
 -Danimal.sniffer.signature=org.codehaus.mojo.signature:java18:1.0 \
 -Danimal.sniffer.failOnError=false

# 如果有有条件判断才会执行的方法，class 上则可以通过增加 @IgnoreJRERequirement 来跳过。


# 需要pom.xml, 构建 自定义 签名.
mvn animal-sniffer:build -DincludeJavaHome=false
# 检查生成的签名文件
ls -l target/*.signature
----


@org.codehaus.mojo.animal_sniffer.IgnoreJRERequirement

[source,xml]
----
<dependency>
  <groupId>org.codehaus.mojo</groupId>
  <artifactId>animal-sniffer-annotations</artifactId>
  <version>1.24</version>
</dependency>
----
