* 如何让特定代码块不进行格式化检查？

link:https://www.jetbrains.com/help/idea/reformat-and-rearrange-code.html#exclude_part_of_code[IdeaIntellij: Exclude code fragments from reformatting in the editor]

link:https://github.com/diffplug/spotless/tree/main/plugin-maven#spotlessoff-and-spotlesson[spotless:off and spotless:on]

[source,xml]
----
<plugin>
    <groupId>com.diffplug.spotless</groupId>
    <artifactId>spotless-maven-plugin</artifactId>
    <version>2.46.1</version>
    <executions>
        <execution>
            <phase>compile</phase>
            <goals>
                <goal>check</goal>
            </goals>
        </execution>
    </executions>
    <configuration>
        <java>
            <toggleOffOn>
                <off>@formatter:off</off>
                <on>@formatter:on</on>
            </toggleOffOn>
            <cleanthat/>
            <removeUnusedImports/>
            <palantirJavaFormat>
                <!-- https://search.maven.org/artifact/com.palantir.javaformat/palantir-java-format -->
                <version>2.68.0</version>
                <style>PALANTIR</style>
                <formatJavadoc>false</formatJavadoc>
            </palantirJavaFormat>
        </java>
    </configuration>
</plugin>
----


[source,java]
----
// 如何让指定代码块不格式化？
// Idea Intellij 通过 `// @formatter:off` + `// @formatter:on`
// spotless 是默认通过 `// spotless:off` + `// spotless:on`, 但建议设置成上面的。

// @formatter:off
here is the java code not check style.
// @formatter:on

----
